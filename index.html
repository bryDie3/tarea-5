Git & GitHub

Hoja de referencia de comandos esenciales

⚙️ Configuración

▼
git config --global user.name "Nombre"
Configura el nombre de usuario global
git config --global user.email "correo@ejemplo.com"
Configura el email global
git config --global init.defaultBranch main
Establece la rama principal por defecto
git config --list
Muestra toda la configuración actual

📁 Iniciar / Clonar Repositorio

▼
git init
Inicializa un repositorio Git local
git clone https://github.com/usuario/repo.git
Clona un repositorio remoto
git clone --depth 1 url
Clona solo el último commit (shallow clone)

👁️ Estado y Cambios

▼
git status
Muestra el estado de los archivos
git status -s
Estado corto y compacto
git diff
Muestra los cambios sin confirmar
git diff --staged
Muestra cambios en el área de staging

📦 Agregar y Confirmar (Stage & Commit)

▼
git add archivo.txt
Agrega un archivo al área de staging
git add .
Agrega todos los archivos modificados
git add -p
Agrega cambios de forma interactiva
git commit -m "Mensaje del commit"
Confirma cambios con un mensaje
git commit -am "Mensaje"
Agrega y confirma cambios (solo para archivos ya seguidos)
git commit --amend -m "Nuevo mensaje"
Modifica el último commit

🌿 Ramas (Branches)

▼
git branch
Lista todas las ramas locales
git branch -a
Lista todas las ramas (locales y remotas)
git branch nueva-rama
Crea una nueva rama
git checkout nueva-rama
Cambia a la rama especificada
git switch nueva-rama
Alternativa moderna para cambiar de rama
git checkout -b nueva-rama
Crea y cambia a una nueva rama
git merge rama-feature
Fusiona la rama especificada con la actual
git branch -d nombre-rama
Elimina una rama (ya fusionada)
git branch -D nombre-rama
Elimina una rama (forzado)

☁️ Repositorios Remotos

▼
git remote -v
Muestra los repositorios remotos configurados
git remote add origin https://github.com/usuario/repo.git
Agrega un repositorio remoto
git remote remove origin
Elimina un repositorio remoto
git push -u origin main
Sube cambios y establece rama upstream
git push origin --delete rama
Elimina una rama remota
git pull origin main
Descarga y fusiona cambios del remoto
git fetch origin
Descarga cambios sin fusionar

📜 Historial de Commits

▼
git log
Muestra el historial de commits
git log --oneline --graph --all
Historial visual en una línea
git log -p -3
Muestra los últimos 3 commits con sus diferencias
git show abc1234
Muestra detalles de un commit específico
git log --author="Nombre" --since="2024-01-01"
Filtra commits por autor y fecha

↩️ Deshacer Cambios

▼
git restore archivo.txt
Restaura archivo al último commit (descarta cambios)
git restore --staged archivo.txt
Saca un archivo del área de staging
git reset HEAD~1
Deshace el último commit (mantiene cambios)
git reset --hard HEAD~1
Deshace el último commit y elimina cambios (cuidado)
git revert abc1234
Crea un commit nuevo que revierte otro (seguro para remoto)
git checkout -- archivo.txt
Alternativa antigua para restaurar archivo

🗂️ Stash (Guardado Temporal)

▼
git stash
Guarda cambios sin confirmar temporalmente
git stash list
Lista todos los stashes guardados
git stash pop
Recupera y elimina el último stash
git stash apply
Recupera el último stash sin eliminarlo
git stash drop stash@{0}
Elimina un stash específico

🏷️ Tags (Etiquetas)

▼
git tag v1.0.0
Crea un tag ligero
git tag -a v1.0.0 -m "Versión 1.0"
Crea un tag anotado con mensaje
git tag
Lista todos los tags
git push origin v1.0.0
Sube un tag al repositorio remoto
git push origin --tags
Sube todos los tags al remoto

🐙 GitHub - Comandos Específicos

▼
gh repo create nombre-repo --public
Crea un nuevo repositorio en GitHub (requiere GitHub CLI)
gh repo clone usuario/repo
Clona un repositorio usando GitHub CLI
gh issue create --title "Bug" --body "Descripción"
Crea un issue desde la terminal
gh pr create --title "Feature" --body "Descripción"
Crea un Pull Request desde la terminal
gh pr list
Lista todos los Pull Requests abiertos
gh pr checkout 123
Descarga y cambia a un PR específico
gh pr merge 123 --merge
Fusiona un Pull Request
gh gist create archivo.txt
Crea un gist desde un archivo
gh auth login
Autentica GitHub CLI en tu cuenta

🙈 .gitignore

▼
touch .gitignore
Crea un archivo .gitignore
git rm -r --cached .
Elimina archivos del índice respetando .gitignore
echo "node_modules/" >> .gitignore
Agrega una regla al .gitignore

💡 Tips Rápidos

▼
git alias
Crea atajos personalizados: git config --global alias.co checkout
git log --all --decorate --oneline --graph
Visualización completa y decorada del historial
git cherry-pick abc1234
Aplica un commit específico a la rama actual
git rebase main
Reorganiza commits sobre la rama main
git bisect start
Inicia búsqueda binaria para encontrar un commit con bug
